# yph.IKeepAccountWeb
>亲密记，由uni-app、Vue3构建。提供账单记录和查看、收支计划管理及进度实时查看等服务。
>
>后端接口使用.NET6 + MongoDB + Redis + RabbitMQ 实现，由Consul提供服务注册发现，由Ocelot提供网关服务。定时调度任务由Hangfire提供支持。
>
>前端支持H5、APP、微信小程序端运行。
